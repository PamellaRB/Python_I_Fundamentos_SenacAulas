1- Crie um programa que peça ao usuário para digitar um número inteiro. O programa deve verificar se o número é par ou ímpar. Se o número for par, exiba a mensagem "O número é par". Caso contrário, exiba "O número é ímpar".

2 - Crie um programa que peça ao usuário para digitar sua idade. O programa deve verificar se o usuário é maior de idade (18 anos ou mais). Se for, exiba a mensagem "Você é maior de idade". Caso contrário, exiba "Você é menor de idade".
